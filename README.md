STEPS TO RUN THE APP:-

INSTALL NODE JS
INSTALL MY SQL SERVER
INSTALL NPM
NPM INIT
NPM INSTALL MYQL
NPM INSTALL FORMIDABLE

1.RUN MAIN.JS TO RUN THE MAIN PROGRAM(NODE main.js)
